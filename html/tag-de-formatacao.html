Elementos Semânticos do HTML5

  • <header>
  • <main>
  • <aside>
  • <footer>
  • <section>
  • <article>
  • <figure>

Header

O elemento <header> é usado para identificar a seção de cabeçalho de um documento ou de uma seção específica. Geralmente, contém elementos como logotipos, títulos, menus de navegação e outras informações importantes para o usuário. Cada página deve conter apenas um elemento <header>, que geralmente está localizado no topo da página.

Main

O elemento <main> é usado para envolver o conteúdo principal de uma página. Cada página deve conter apenas um elemento <main>. Isso ajuda a fornecer um foco claro no conteúdo principal, tornando a navegação e a acessibilidade mais fáceis para os usuários.

Exemplo:

Aside

O elemento <aside> é utilizado para definir conteúdo relacionado, mas que é considerado secundário em relação ao conteúdo principal. Isso pode incluir informações adicionais, anúncios, links relacionados e muito mais. O conteúdo dentro de <aside> geralmente está separado do conteúdo principal.

Exemplo:

Footer

O elemento <footer> é usado para definir o rodapé de um documento ou de uma seção específica. Normalmente, ele contém informações como direitos autorais, informações de contato, links para políticas de privacidade e termos de serviço, entre outros.

Exemplo:

Section

O elemento <section> é usado para dividir o conteúdo em seções temáticas ou agrupadas de forma lógica. Cada seção pode ter seu próprio cabeçalho e conteúdo. Ele ajuda a organizar o conteúdo de forma mais significativa e é frequentemente usado em conjunto com elementos de cabeçalho.

Article

O elemento <article> é usado para marcar conteúdo independente, como postagens de blog, notícias, artigos, etc. Cada <article> deve ser autônomo e fazer sentido por si só, mesmo quando removido do contexto da página.
